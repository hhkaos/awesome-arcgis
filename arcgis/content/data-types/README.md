## Introduction
In addition to [geodatabases](../data-storage/odatabase/README.md), a fundamental goal of ArcGIS is to work with numerous file-based data types; DBMS dataâ€”including spatial types in the DBMS; and GIS web services from Google, Bing, ArcIMS, and ArcGIS for Server and others such as OGC web services. This topic provides an overview about many of the foreign spatial data formats supported by ArcGIS.

ArcGIS lets you work with a number of data formats from many sources. Some of the supported data sources are listed here.
